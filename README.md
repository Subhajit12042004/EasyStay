## 📌 Overview

**EasyStay** is a full-stack web application that allows users to explore, create, and manage property listings.
It provides a simple interface for adding listings and reviews, built using Node.js, Express, and MongoDB.

---

## 🚀 Features

* 🏠 Create, edit, and delete property listings
* 📄 View detailed listing information
* ⭐ Add and manage reviews
* 🔄 RESTful routing (CRUD operations)
* ✅ Server-side validation using Joi
* ⚠️ Centralized error handling

---

## 🧠 Tech Stack

* **Frontend:** EJS, HTML, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Validation:** Joi
* **Other:** Method-Override, EJS-Mate

---

## ⚙️ How It Works

1. User interacts with frontend (EJS templates)
2. Express handles routing and logic
3. MongoDB stores listings and reviews
4. Joi validates user input
5. Data rendered dynamically

## 🛠️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/EasyStay.git
cd EasyStay
```

Install dependencies:

```bash
npm install
```

Run the server:

```bash
node app.js
```

Open in browser:

```
http://localhost:8080
```
